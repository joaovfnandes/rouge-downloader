ROUGE PORTÁTIL v1.1.2
======================

1. Extraia o ZIP inteiro antes de abrir.
2. Abra Rouge.exe com dois cliques.
3. O site será aberto automaticamente no navegador.
4. Deixe a janela preta do Rouge aberta enquanto estiver usando o programa.
5. Para encerrar completamente, feche essa janela.

NOVIDADES DA v1.1.2
-------------------
- Filtro de tags com expansão da lista.
- Filtro da coleção por título, código ou tag.
- Layouts Editorial, Mosaico e Foco para as imagens recentes.

SEUS DADOS
----------
- Sua coleção fica na pasta nhentai_out.
- As tags descobertas ficam no arquivo tag_history.json.
- Os registros ficam na pasta logs.

Para levar o programa a outro computador Windows, copie a pasta Rouge inteira.
Não mova apenas Rouge.exe, pois a pasta _internal também é necessária.

Leia GUIA_PORTATIL.md para instruções completas.
